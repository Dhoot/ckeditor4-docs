Exporting content to a PDFDocumentation

Despite CKEditor 4 operates on HTML and is basically a web tool, you are able to save it's content to a file and use it elsewhere. This is possible thanks to the optional Export to PDF plugin which allows saving the editor content as PDF.

It introduces a  new toolbar button. When it's clicked, data from editor is sent to the service maintained by CKEditor Cloud Services. The editor takes care of the received response and saves it as a file.

Internet Explorer browser support is limited: the plugin is only available for version 11. If you need to use it in the older one, please contact us.
